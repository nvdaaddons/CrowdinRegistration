# Копіювання курсору перегляду (Review Cursor Copier)

* Автор: Tuukka Ojala <tuukka.ojala@iki.fi>
* Завантажити [стабільну версію][1]
* Завантажиди [версію для розробників][2]

Цей NVDA додаток надає різноманітні команди для копіювання тексту під курсором перегляду до буфера обміну. Наразі реалізовано такі команди:

* Скопіювати рядок під курсором перегляду
* Скопіювати слово під курсором перегляду
* Скопіювати від початку поточного рядка до курсору перегляду
* Скопіювати від курсору перегляду до кінця поточного рядка
* Скопіювати від початку поточного слова до курсору перегляду
* Скопіювати від курсору перегляду до кінця поточного слова

Для жодної з цих команд не призначено клавіш за замовчуванням. Будь ласка, скористайтеся діалоговим вікном «Жести вводу», що знаходиться в меню параметрів NVDA, щоб призначити їх. Усі команди, які надає цей додаток, можна знайти в категорії «Перегляд тексту». Більше інформації про призначення та зміну жестів вводу можна знайти в посібнику користувача NVDA.
