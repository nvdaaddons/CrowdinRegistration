This [NVDA](https://nvaccess.org/) add-on adds a shortcut to announce the name, version, and now the architecture of the foreground application, or copies this information to the clipboard.

* English keyboard shortcut: `NVDA+shift+v`
* Press it once to hear the information spoken, or receive it in braille.
* Press it twice to copy name, version, and architecture to the clipboard, separated by newlines.
* Press it three times to copy only the version and architecture to the clipboard.

Эта горячая клавиша может быть переназначена в категории сервиса жестов ввода NVDA.

To access the Windows version, use the shortcut on the Desktop or Taskbar.
It will also work from various other parts of Windows.

To access the NVDA version, use the shortcut while the NVDA menu, add-on store, or similar NVDA interface is open.
