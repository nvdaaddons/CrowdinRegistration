# Віртуальний огляд

* Автори: Rui Fontes, Rui Batista та NVDA Addon Team
* Завантажити [стабільну версію][1]
* Завантажити [версію розробки][2]
* Сумісність з NVDA: 2019.3 та новіші

Цей додаток дозволяє користувачам NVDA переглядати вміст вікна у текстовому полі, схоже на віртуалізацію вікон у JAWS для Windows.
Утім, майте на увазі: ця функція є лише допоміжною і не є заміною для потужних режимів перегляду та об’єктної навігації NVDA.

## Використання ##

Натисніть NVDA+control+w, щоб відкрити вікно віртуального перегляду.
Потім просто переміщуйтеся по показаному текстовому полі так само, як у будь-якому іншому текстовому вмісті.
Ви можете натиснути Escape, щоб закрити вікно віртуального перегляду.

## Зміни в 3.0

* Вимагає NVDA версії 2019.3 або новішої.

## Зміни в 2.3

* Внутрішні зміни для підтримки майбутніх випусків NVDA.

## Зміни в 2.2

* Коли вікна згорнуті або якщо вікно на передньому плані не має заголовка, NVDA зафіксує це під час показу вікна віртуального перегляду.

## Зміни в 2.1

* Включатиметься заголовок екрана, що перебуває під віртуальним переглядом.
* Тепер можливо переглядати вміст Universal Windows Platform (UWP) додатків у Windows 10 та більшість сучасних додатків у Windows 8.x.
* Під час отримання вмісту екрана для сучасних та універсальних додатків прозвучить звуковий сигнал, який вказує, що вікно віртуального перегляду готове.

* Виправлено функціональність додатка з не-консольними вікнами.
