# Virtuaalitarkastelu

* Tekijät: Rui Batista ja NVDA-lisäosatiimi
* Lataa [vakaa versio][1]
* Lataa [kehitysversio][2]
* Yhteensopivuus: NVDA 2019.3 ja uudemmat

Tämän lisäosan avulla NVDA-käyttäjät voivat tarkastella ikkunan sisältöä tekstiruudussa samaan tapaan kuin JAWSin ikkunan virtualisointitoiminnolla.
Ota kuitenkin huomioon, että tämä on vain käyttömukavuutta lisäävä toiminto, eikä se korvaa NVDA:n tarkastelutiloja tai objektinavigointia.

## Käyttö ##

Virtualisoi aktiivinen ikkuna painamalla NVDA+Ctrl+W.
Voit liikkua näkyviin tulevassa tekstikentässä kuten missä tahansa muussakin tekstimuotoisessa sisällössä.
Sulje ikkuna painamalla Esc-näppäintä.

## Muutokset versiossa 3.0

* Edellyttää NVDA 2019.3:a tai uudempaa.

## Muutokset versiossa 2.3

* Sisäisiä muutoksia tulevien NVDA-versioiden tukemiseksi.

## Muutokset versiossa 2.2

* Kun ikkunat on pienennetty tai etualan ikkunalla ei ole nimeä, NVDA ottaa sen huomioon avatessaan virtuaalitarkastelun ikkunaa.

## Muutokset versiossa 2.1

* Ikkunan nimi näytetään virtuaalitarkastelussa.
* Windows 10:n Universal Windows Platform (UWP)- sekä useimpien Windows 8.x:n modernien sovellusten tarkastelu on nyt mahdollista.
* Merkkiääni ilmaisee moderneissa ja universaaleissa sovelluksissa ruudun sisältöä noudettaessa, kun virtuaalitarkastelun ikkuna on valmiina.

## Muutokset versiossa 2.0

* Lisäosan uusi nimi on Virtuaalitarkastelu.
* Sisällön näyttämiseen tehtyjen muutosten vuoksi edellytetään NVDA 2015.2:ta tai uudempaa.
* NVDA 2015.2:n ja sitä uudempien versioiden uusien ominaisuuksien ansiosta tekstin etsiminen on nyt mahdollista virtualisoiduissa ikkunoissa.
* Korjattu lisäosan toiminnallisuus muissa kuin konsoli-ikkunoissa.
* Poistettu turhat tyhjät rivit virtualisoidun konsoli-ikkunan alaosasta.
* Tehty joitakin pieniä muutoksia.
* Käännöksiä lisätty.

## Muutokset versiossa 1.2

## Muutokset versiossa 1.1

## Muutokset versiossa 1.0
